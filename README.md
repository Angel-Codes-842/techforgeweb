# TechForge Solutions - Landing Page

Este es el repositorio para la landing page de TechForge Solutions, creada con React, Vite y TailwindCSS.

## Requisitos

- Node.js (v18 o superior)
- npm

## Cómo ejecutar el proyecto

1.  **Clona el repositorio (si aplica):**
    ```bash
    git clone <URL_DEL_REPOSITORIO>
    cd <NOMBRE_DEL_DIRECTORIO>
    ```

2.  **Instala las dependencias:**
    ```bash
    npm install
    ```

3.  **Inicia el servidor de desarrollo:**
    ```bash
    npm run dev
    ```

El sitio estará disponible en `http://localhost:5173` (o el puerto que indique Vite).
